Temaet benytter sig af tredjepartsplugins, som gør det nemt at installere 
ekstra indholdstyper og navigere herimellem. Temaet tager for givet at de 
nødvendige plugins er installeret. Herunder er en liste over de anvendte 
plugins, så det er nemt at finde dem, hvis de afinstalleres ved en fejl.

Vi benytter IKKE "Must Use", fordi vi gerne vil sikre os at plugins holdes
opdateret. (se evt. http://codex.wordpress.org/Must_Use_Plugins)

Sørg altid for at foretage backup inden opdatering af core eller plugins. 
Både MySQL-database og filer på serveren. Så kan du altid rulle 
opdateringer tilbage, hvis noget går galt. Som udgangspunkt skulle 
opdatering af core eller plugins dog være ganske ufarligt :-)


/* [1.1] : Meta Box (Af Rilwis) ----------------------------------------*/

Meta Box er en udvikler-ekstension som gør det meget nemt at tilføje 
post_meta's til enkelte eller flere post types. se en fuld demo af 
tilgængelige meta-felter på
https://github.com/rilwis/meta-box/blob/master/demo/demo.php

Projektside: http://www.deluxeblogtips.com/meta-box/
Pluginside: https://wordpress.org/plugins/meta-box/
